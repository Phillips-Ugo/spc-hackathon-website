What We Built

A full-stack robotics toolkit for the Seeed B601-DM follower arm. We built scripts to calibrate, teleoperate, jog individual joints, move with URDF/Pinocchio-based inverse kinematics, probe pen-to-paper contact using torque sensing, and autonomously draw letters and words — all orchestrated through the LeRobot framework.

The system uses dynamic contact detection by monitoring elbow_flex.torque and shoulder_lift.torque to find the paper surface in real time, eliminating hard-coded magic Z values. Once contact is detected, the arm begins drawing immediately, lifting for pen-up travel between strokes and returning to teleop home when finished.

Real-Time Teleoperation

A leader arm mirrors every joint movement to the follower in real time. The operator physically moves the leader while the follower replicates each motion, enabling intuitive and precise remote manipulation.

Use Cases

Automated Handwriting & Precision Drawing

Manufacturing labels, PCB trace marking, or personalised engravings — the arm converts text into multi-stroke waypoints and draws them on any flat surface. Dynamic contact sensing adapts to surface height variations, so the system works on curved or uneven materials without manual recalibration. New names or shapes can be added in minutes by editing 2D stroke coordinates.

Remote & Hazardous-Environment Teleoperation

Using the leader–follower architecture, an operator can manipulate objects from a safe distance — handling radioactive samples, performing cleanroom assembly, or assisting in surgical pre-positioning. The low-latency joint mirroring and torque feedback loop give the operator a natural feel, while the software safety layer prevents over-torque and out-of-bound motions.
